ticino - events, not elephants

Welcome!

ticino is a tiny type-safe java event framework with a minimalistic class interface.

ticino introduces the silent broker approach:

  • as few as possible code changes

  • as few as possible design decisions in client code towards ticino

  • as few as possible transitive dependencies

  • as few as possible side effects

  • uses the MIT license - no license, copyright or copyleft problems at all

  • being fully compatible with other event frameworks

See the cookbook for examples and best practices

Downloading ticino events with maven


<dependencies>
    ...
	<dependency>
		<groupId>net.micwin.ticino</groupId>
		<artifactId>ticino-events</artifactId>
		<version>0.3.4</version>
	</dependency>
	...
</dependencies>
